## 目標佈局：三欄式全螢幕

```
┌─────────────────────────────────────────────────────────────────┐
│  Top Bar (48px)                                                 │
│  [Logo / 標題]          [上傳] [AI 操作] [影片資訊 ℹ]    [⚙]   │
├────────────┬───────────────────────────────────┬────────────────┤
│ 左側欄      │         中央畫布                   │ 右側欄          │
│ (240px)    │    (flex:1, 吃滿剩餘空間)           │ (300px)        │
│            │                                   │                │
│ ┌────────┐ │  ┌───────────────────────────┐    │ ┌────────────┐ │
│ │ 影片   │ │  │                           │    │ │ Category   │ │
│ │ 列表   │ │  │     影片 + Overlay         │    │ │ Layers     │ │
│ │        │ │  │     (深色背景)              │    │ ├────────────┤ │
│ │ 🟢 v1  │ │  │                           │    │ │ Annotation │ │
│ │ 🟡 v2  │ │  │                           │    │ │ Layers     │ │
│ │ 🔴 v3  │ │  └───────────────────────────┘    │ ├────────────┤ │
│ └────────┘ │  ┌───────────────────────────┐    │ │ AI Layers  │ │
│            │  │ ◁  ▶  ▷ │ 00:03 / 00:13 │    │ └────────────┘ │
│            │  │ ───●──────────── │ 1x ▾   │    │                │
│            │  └───────────────────────────┘    │                │
├────────────┴───────────────────────────────────┴────────────────┤
│  Status Bar (24px)  3840×2160 · 23.97fps · f:001 · pts:0       │
└─────────────────────────────────────────────────────────────────┘
```

### 佈局原則

- **中央畫布佔螢幕 60–70%**，是使用者主要工作區域
- 左右側欄可透過拖曳調整寬度，也可點擊折疊
- Top Bar 高度固定 48px，僅放全域操作
- Status Bar 高度固定 24px，顯示影片 metadata 摘要
- 全部區域加總 = `100vh`，不產生頁面捲軸

#### 6-A. AI Layers 列表行設計（唯讀）

AI 結果來自 `latest.coco.json`，為唯讀資料，**不提供 delete**。

```
┌──────────────────────────────────────────────────┐
│ ▾ AI Layers                               2 bbox │
│  ☑ BBox   ☑ Track ID   ☐ Trajectory             │
│  Confidence ──●──── 50%  (篩選門檻)               │
├──────────────────────────────────────────────────┤
│  #1  ■ car  [120, 80, 340, 220]  92%   👁   📋   │
│  #2  ■ car  [580, 160, 90, 70]   83%   👁   📋   │
└──────────────────────────────────────────────────┘
```

| 欄位 | 說明 |
|------|------|
| `#1` | track_id（有 track 時）或流水號 |
| `■ car` | 類別色塊 + 名稱 |
| `[x,y,w,h]` | bbox 座標（整數像素），click 可複製 |
| `92%` | 信心值純文字 |
| `👁` | 單筆可見開關 — 隱藏該 bbox overlay（不改 COCO 資料） |
| `📋` | 複製為手動標註 — 以此 bbox 為模板建立 `source=MANUAL` annotation |

互動規則：

1. ✅ hover 列表項 → 畫布對應 bbox 高亮（白色外框 + 橘色陰影）
2. ✅ click 列表項 → 選中狀態（橘色邊框 + 淡色背景），雙向同步
3. ✅ click 畫布 bbox → 列表自動捲動並選中對應項
4. ✅ Confidence slider → 即時過濾低於門檻的 bbox（list + canvas 同步）

#### 6-B. Annotation Layers 列表行設計（完整 CRUD）

人工標註為使用者建立的資料，**提供完整 CRUD 含 delete**。

```
┌──────────────────────────────────────────────────┐
│ ▾ Annotation Layers                       1 bbox │
├──────────────────────────────────────────────────┤
│  ■ nerve  [120, 80, 340, 220]       👁  ✏️  🗑   │
└──────────────────────────────────────────────────┘
```

| 動作 | 說明 |
|------|------|
| `👁` | 隱藏 / 顯示該筆標註 |
| `✏️` | 編輯 — 畫布出現拖曳把手可調整 bbox，或展開行內編輯 x/y/w/h |
| `🗑` | 刪除（需確認對話框） |

互動規則：

1. ✅ hover / click 聯動：列表列選中 ↔ 畫布高亮雙向同步
2. ✅ 編輯模式：bbox 8 個 resize 把手 + 拖曳移動，polygon 頂點拖曳
3. ✅ 新增標註：工具列選 RECT/POLYGON/TEXT → 畫布拖曳建立 → 自動儲存

## 技術選型摘要

| 面向 | 選擇 | 理由 |
|------|------|------|
| CSS 框架 | Tailwind CSS | Next.js 官方支援、utility-first、開發快速 |
| 元件庫 | shadcn/ui (Radix + Tailwind) | 可挑選元件、不臃腫、完全可自訂樣式 |
| Icon | Lucide React | shadcn/ui 預設搭配、輕量、風格一致 |
| 面板拖曳 | react-resizable-panels | 輕量、React 友好、支援持久化尺寸 |
| 深色主題 | Tailwind `darkMode: 'class'` + CSS 變數 | 標準作法、未來可支援亮/暗切換 |
